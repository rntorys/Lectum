# Lectum
Lectum es un dashboard academico para gestionar materias y notas, con modos de calculo (porcentaje, promedio y geometrica), notas y controles, programacion de pruebas futuras, y filtros por grupo/semestre. Incluye modo claro/oscuro, promedios generales, alertas por porcentaje acumulado y almacenamiento local para conservar los datos.
